afo.video是一个基于solon，Sa-Token，MyBatis-Flex，MySQL的简单视频平台后端。
使用solon框架，使用solon插件。
murky是基于相同技术的脚手架，solon-examples是基于solon的示例项目。 后端代码风格和规范参考二者。

在终端构建或运行后端项目时，需要先设置环境变量:
```
$env:JAVA_HOME='C:\Users\Administrator\jdk\jdk-17.0.12'; $env:PATH='C:\Users\Administrator\jdk\jdk-17.0.12\bin;' + $env:PATH; java -version
```

frontend是基于vue、vite、element-plus、ts、vue-router、axios、pinia的前端文件夹
前端命令使用pnpm，终端命令使用powerShell。

修改代码时尽量不改变后端代码，修改前端代码时要对齐后端代码的规范。
